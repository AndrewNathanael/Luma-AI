Gambar 2 — Sub-proses Fase Computer Vision
Langkah Deskripsi Detail Implementasi
Pemetaan Landmark Mendeteksi 478 titik landmark wajah pada setiap frame MediaPipe FaceLandmarker (float16), min_detection_confidence = 0.5
Isolasi ROI Membuat 3 region polygon dari subset landmark tertentu Dahi: 12 titik boundary, Pipi Kiri: 9 titik, Pipi Kanan: 9 titik (mirror simetris)
Skin Filtering Memfilter hanya piksel kulit valid dalam setiap region Konversi BGR → YCrCb, threshold: Y: 0–255, Cr: 133–173, Cb: 77–127
Rata-Rata Spasial Menghitung nilai rata-rata R, G, B dari piksel kulit valid Gabungan mask polygon + skin; fallback ke seluruh polygon jika piksel kulit < 10

Gambar 3 — Sub-proses Fase Pemrosesan Sinyal
Langkah Deskripsi Detail Implementasi
Algoritma POS Plane-Orthogonal-to-Skin — memproyeksikan sinyal RGB ke bidang ortogonal terhadap variasi warna kulit Matriks proyeksi [[0, 1, -1], [-2, 1, 1]], alpha adaptif berdasarkan rasio standar deviasi
Overlap-Add Menggabungkan hasil proyeksi dari window kecil yang saling tumpang tindih Window = 1.6 detik, step = 1 frame, akumulasi dengan pembobotan
Bandpass Filter Mempertahankan hanya komponen frekuensi detak jantung (42–240 BPM) Butterworth order 4, zero-phase (sosfiltfilt), range: 0.7 – 4.0 Hz

Gambar 5 — Pipeline Machine Learning

Tabel Output Klasifikasi

Kelas Label Kondisi Asli Deskripsi
0 Normal Rest (T1) Kondisi istirahat, baseline fisiologis
1 Moderate Arousal Speech (T2) Stres ringan–sedang akibat tugas berbicara
2 High Arousal Arithmetic (T3) Stres tinggi akibat tugas aritmatika mental

Gate Keputusan Output

Gate Threshold Jika Gagal
Kualitas Sinyal > 0.60 → "Poor Signal" (ulangi akuisisi)
Detection Rate ≥ 0.90 → "Face Not Stable" (reposisi wajah)
Confidence + Quality Confidence ≥ 0.60 DAN Quality ≥ 0.70 → "Uncertain" (hasil + peringatan)

Validasi Model

Metode Deskripsi
Subject-Independent Split Train/Validation/Test dipisah berdasarkan subjek (tidak ada subjek yang overlap)
Leave-One-Subject-Out (LOSO) Setiap subjek dijadikan test set secara bergantian untuk evaluasi yang lebih robust
